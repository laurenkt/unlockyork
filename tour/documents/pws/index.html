Attribute Group: position

The position and size (offset) of an element. The width and height of an element is defined as the difference between the (x, y) and (x2, y2) coordinates.
Attributes
x Built-in type decimal optional
The x co-ordinate of the top left corner of the element
y Built-in type decimal optional
The y co-ordinate of the top left corner of the element
x2 Built-in type decimal optional
The x co-ordinate of the bottom right corner of the element
y2 Built-in type decimal optional
The y co-ordinate of the bottom right corner of the element

Attribute Group: fonts

All text formatting options (except color). If these attributes are omitted on an element, it will inherit them from its parent
Attributes
font Built-in type string optional
The font family to be used. If the specified family is not available, the implementor may define a fallback.
italic Built-in type boolean optional
Whether the text should be italic.
bold Built-in type boolean optional
Whether the text should be bold.
textsize Built-in type int optional
The size (in points)
underline Built-in type boolean optional
Whether the text should be underlined.

Attribute Group: colors

Defines the fill color and the foreground/stroke/text color
Attributes
color Simple type colorType optional
The foreground/stroke/text colour.
fill Simple type colorType optional
The background colour.

Attribute Group: transitions

Timing controls for automatic animation/transitions of elements
Attributes
start Simple type startType optional
How long in seconds before the element appears, or whether it should wait to be user-triggered
duration Built-in type int optional
How long in seconds before the element disappears

Simple Type: startType

Either an integer number of seconds specifying how long to wait until starting the transition, or the word "trigger" which specifies waiting for an event (such as a mouse click)

Built-in type string ( trigger )

OR

Built-in type integer

Simple Type: shapeType

Defines all vector shapes that are supported.

Built-in type string ( ellipse,rectangle,line )

Simple Type: colorType

Defines supported color formats; Hash prefixed 6-digit hex codes, in the form of '#RRGGBB'. Two color vertical gradients, in the form of 'gradient(#RRGGBB,#RRGGBB)'. Alpha is not supported, nor are 3-digit hex codes.

Built-in type string

Pattern: #[0-9A-Fa-f]{6}

OR

Built-in type string

Pattern: gradient\(#[0-9A-Fa-f]{6},#[0-9A-Fa-f]{6}\)
